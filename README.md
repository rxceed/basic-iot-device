# basic-iot-device
Respository untuk project basic IoT device mata kuliah Jaringan Komputer

# Alat dan Bahan
- Raindrop sensor
- DHT11 sensor
- ESP32
- Kesabaran

## Back end install
```
cd ./back-end
npm install
```
## Front end install
```
cd ./front-end
npm install
```
